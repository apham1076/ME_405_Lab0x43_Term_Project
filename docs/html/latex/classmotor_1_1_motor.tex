\doxysubsection{Detailed Description}
\begin{DoxyVerb}A motor driver interface encapsulated in a Python class. Works with
   motor drivers using separate PWM and direction inputs such as the DRV8838
   drivers present on the Romi chassis from Pololu.\end{DoxyVerb}
