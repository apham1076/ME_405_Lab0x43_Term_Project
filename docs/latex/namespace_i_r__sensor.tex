\begin{DoxyVerb}@file IR_sensor.py
@brief IR sensor array driver with per-channel calibration and centroid calculation.

This module defines an IRArray class that manages an array of IR sensors. It supports per-channel calibration for black and white surfaces, single-shot and timed multi-sample readings, and centroid calculation for line following applications.\end{DoxyVerb}
